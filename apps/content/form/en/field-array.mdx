---

`FieldArray` is a utility component that uses the function-as-children pattern to make working with [react-hook-form](https://react-hook-form.com/) field arrays simple and ergonomic inside a shaddy form.  
It provides a convenient way to render, append, and remove groups of fields dynamically, without having to manage the array logic yourself.

### Usage

```tsx
<FieldArray name="items">
  {({ fields }) =>
    fields.map((field, index) => (
      <TextField key={field.id} name={`items.${index}.name`} />
    ))
  }
</FieldArray>
```
